# iotera-site

Static marketing and landing site for IoTera.

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Notes

- Intended deploy target: Netlify
- Intended canonical domain: `iotera.asia`
- Product subdomains remain separate:
  - `app.iotera.asia`
  - `api.iotera.asia`
  - `mqtt.iotera.asia`
